MindShelf

iOS + web / in production

A book tracking product with a native iOS app and a companion web app - both live in production and sharing the same account.

  • iOS app with in-app purchases, paywall, and restore purchases via StoreKit 2
  • Supabase for auth and real-time sync between iOS and the web
  • Web app on Cloudflare Pages with metadata auto-fetch via microlink and noembed
  • Full free-tier, paywall, and monetization flow shipped end to end

// Swift / SwiftUI / StoreKit 2 / Supabase / React / Vite / Cloudflare Pages
